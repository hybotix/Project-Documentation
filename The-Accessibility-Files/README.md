# The Accessibility Files

## Overview

The Accessibility Files is a Hybrid RobotiX initiative focused on accessibility technology research, tooling, and advocacy. It reflects Dale's lived experience as a power wheelchair user and commitment to making technology genuinely accessible.

## Status

🟢 Ongoing — scope and active deliverables being defined

## Areas of Focus

- Accessibility feature contributions to open-source projects
- Documentation and research on assistive technology
- Advocacy for accessibility-first design
- Dark Reader compatibility and display accessibility

## Notable Contributions

- Dark Reader compatibility features contributed to Delta Robotics Protoboard PCB design software (as Alpha User)
