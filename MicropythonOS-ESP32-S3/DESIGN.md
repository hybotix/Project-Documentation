# MicropythonOS / ESP32-S3 — Design

## MicropythonOS

MicropythonOS is an Android-inspired operating system for ESP32 devices, written entirely in MicroPython. It provides an app model, launcher, and runtime environment on microcontrollers.

## Target Board

Waveshare ESP32-S3-Touch-LCD-2 is the recommended board for MicropythonOS, providing:
- ESP32-S3 SoC with PSRAM
- 2" capacitive touch LCD
- USB-C connectivity

## Scope

This is a personal tinkering project. No integration with other Hybrid RobotiX systems is planned at this time.
