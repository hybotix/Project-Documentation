# MicropythonOS / ESP32-S3

## Overview

Personal tinkering project exploring MicropythonOS — an Android-inspired operating system for ESP32 written in MicroPython — on a Waveshare ESP32-S3-Touch-LCD-2 board.

## Status

🟢 Hardware received; initial exploration phase

## Board

| Field | Detail |
|---|---|
| Board | Waveshare ESP32-S3-Touch-LCD-2 |
| Amazon ASIN | B0DTTL56ZR |
| SoC | ESP32-S3 |
| Display | 2" capacitive touch LCD |

## Goals

- Explore MicropythonOS app model (Android-inspired architecture)
- Personal tinkering — not tied to a larger Hybrid RobotiX project
- Understand MicroPython capabilities on ESP32-S3 with integrated display
